# eth-read-api

## Getting Started

### Install dependencies

```shell
$ npm install
```

### Run dev server

```shell
$ npm run dev
```

### Run tests

```shell
$ npm run test
```

## Notable dependencies

- `express`: web framework
- `ethers`: for interacting with the blockchain
- `node-cache`: to cache ABI responses from etherscan
- `supertest`: for writing e2e tests
