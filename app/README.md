# eth-read-app

## Getting started

### Install dependencies

```shell
$ npm install
```

### Run dev server

```shell
$ npm run dev
```

### Lint

Ideally should be set up in IDE, but can be run independently as well.

```shell
$ npm run clean
```

## Notable dependencies

- `@tanstack/react-query`: for handling API calls
- `tailwindcss`: for styling components
